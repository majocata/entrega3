¿Que es excel?
Microsoft Excel es una hoja de cálculo desarrollada por Microsoft para Windows, macOS, Android e iOS. Cuenta con cálculo, herramientas gráficas, tablas calculares y un lenguaje de programación macro llamado Visual Basic para aplicaciones.
¿Para que sirve?
Podemos decir que Excel es una herramienta para poder hacer cualquier tipo de cálculo matemático en una hoja: hoja de cálculo. Se pueden realizar operaciones matemáticas y crear tablas de números para realizar operaciones con ellas posteriormente de una manera eficaz y rápida
